Accessibilité du site

Introduction

Nous nous engageons à rendre ce site accessible à tous les utilisateurs, y compris les personnes en situation de handicap. Voici les principales mesures d’accessibilité mises en place :

Fonctionnalités d’accessibilité

  • Navigation clavier : Toutes les fonctionnalités du site sont accessibles au clavier (tabulation, entrée, etc.).
  • Contrastes élevés : Les couleurs et contrastes sont choisis pour garantir une bonne lisibilité.
  • Textes alternatifs : Toutes les images importantes disposent d’un texte alternatif (alt).
  • Structure sémantique : Utilisation appropriée des balises HTML (titres, listes, paragraphes) pour faciliter la navigation avec les lecteurs d’écran.
  • Labels explicites : Les champs de formulaire sont associés à des labels clairs.
  • Zoom et taille du texte : Le site reste lisible et fonctionnel même avec un zoom important ou une taille de police modifiée.
  • Liens explicites : Les liens sont compréhensibles hors contexte.
  • Notifications accessibles : Les messages d’erreur et de confirmation sont annoncés aux technologies d’assistance.

Compatibilité

Ce site est compatible avec les principaux navigateurs et lecteurs d’écran.

Contact

Si vous rencontrez un problème d’accessibilité, merci de nous contacter via notre page de contact pour que nous puissions l’améliorer.
